API Project: URL Shortener Microservice

Welcome to my project!

Send an url of the form http(s)://www.yourpage.com to my server and you'll receive a shortened version for easier access.

For example, if you receive a shortened url of 10, you can access it like this:
https://gabrieljim-urlshortener.glitch.me/api/shorturl/10
And it'll redirect you
